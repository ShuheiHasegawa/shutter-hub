# ShutterHub ページ機能一覧

ShutterHubは、撮影業界の三者（モデル、カメラマン、運営者）をつなぐ統合型プラットフォームです。このドキュメントでは、各ページでユーザーができることをページ単位でまとめています。

---

## 📱 認証・アカウント管理

### `/auth/signin` - ログインページ
- **できること**:
  - 既存アカウントでログインする
  - OAuth認証（Google、X、Discord）でログインする
  - パスワードリセットをリクエストする

### `/auth/signup` - 新規登録ページ
- **できること**:
  - 新規アカウントを作成する
  - OAuth認証で新規登録する
  - ユーザータイプ（モデル、カメラマン、運営者）を選択する

### `/auth/onboarding` - オンボーディングページ
- **できること**:
  - 初回ログイン時のプロフィール情報を設定する
  - 基本情報（表示名、自己紹介等）を入力する

### `/auth/setup-profile` - プロフィール設定ページ
- **できること**:
  - プロフィール情報を詳細に設定する
  - アバター画像をアップロードする
  - 活動拠点を設定する

### `/auth/auth-code-error` - 認証エラーページ
- **できること**:
  - 認証コードエラー時の対処方法を確認する
  - 再認証を試みる

---

## 🏠 ホーム・ランディング

### `/` - ホームページ（ランディングページ）
- **できること**:
  - ShutterHubの概要と機能を確認する
  - 撮影会検索ページへ遷移する
  - 即座撮影リクエストページへ遷移する
  - 新規登録ページへ遷移する
  - 主要機能（撮影会予約、即座撮影、StudioWiki等）を確認する

### `/lp-v1`, `/lp-v2`, `/lp-v3` - ランディングページ（バリエーション）
- **できること**:
  - 異なるデザインのランディングページを確認する
  - A/Bテスト用のバリエーションを閲覧する

## 📅 撮影会関連

### `/photo-sessions` - 撮影会一覧ページ
- **できること**:
  - 撮影会を検索・フィルタリングする（キーワード、場所、価格、日付、予約タイプ等）
  - 活動拠点に基づいて自動フィルタリングする
  - ソート順を変更する（開始日時、価格、作成日、人気度等）
  - レイアウトを切り替える（グリッド/リスト表示）
  - 無限スクロールで撮影会を閲覧する
  - お気に入りに追加する
  - 撮影会詳細ページへ遷移する

### `/photo-sessions/[id]` - 撮影会詳細ページ
- **できること**:
  - 撮影会の詳細情報を確認する（タイトル、説明、日時、場所、価格等）
  - 予約可能なスロットを確認する
  - 予約を申し込む（先着順、抽選、優先予約、キャンセル待ち）
  - 運営者情報を確認する
  - レビューを閲覧する
  - お気に入りに追加/削除する
  - 参加者一覧を確認する（運営者の場合）

### `/photo-sessions/[id]/edit` - 撮影会編集ページ
- **できること**:
  - 撮影会情報を編集する（運営者のみ）
  - スロット情報を追加・編集・削除する
  - 予約タイプを変更する
  - 価格を変更する
  - 撮影会を公開/非公開にする

### `/photo-sessions/[id]/participants` - 参加者管理ページ
- **できること**:
  - 予約済み参加者一覧を確認する（運営者のみ）
  - 参加者の詳細情報を確認する
  - 抽選結果を確認・管理する
  - キャンセル待ちリストを確認する

### `/photo-sessions/[id]/reviews` - レビュー一覧ページ
- **できること**:
  - 撮影会のレビューを閲覧する
  - レビューを投稿する（参加者のみ）
  - レビューにいいねする

### `/photo-sessions/[id]/analytics` - 撮影会分析ページ
- **できること**:
  - 撮影会の統計情報を確認する（運営者のみ）
  - 予約状況を確認する
  - 参加者の属性を分析する

### `/photo-sessions/[id]/duplicate` - 撮影会複製ページ
- **できること**:
  - 既存の撮影会を複製して新規作成する（運営者のみ）
  - 類似した撮影会を効率的に作成する

### `/photo-sessions/create` - 撮影会作成ページ（選択）
- **できること**:
  - 撮影会作成方法を選択する（個人運営/共同運営）

### `/photo-sessions/create/organizer` - 個人運営撮影会作成ページ
- **できること**:
  - 個人で運営する撮影会を作成する
  - 基本情報（タイトル、説明、日時、場所等）を入力する
  - 価格を設定する
  - 予約タイプを選択する

### `/photo-sessions/create/joint` - 共同運営撮影会作成ページ
- **できること**:
  - 複数の運営者で共同運営する撮影会を作成する
  - 共同運営者を招待する
  - 権限を設定する

### `/photo-sessions/favorites` - お気に入り撮影会一覧
- **できること**:
  - お気に入りに登録した撮影会を一覧表示する
  - お気に入りから削除する

## ⚡ 即座撮影

### `/instant` - 即座撮影リクエストページ
- **できること**:

- **できること**:
  - 決済情報を入力する

### `/instant/payment/[bookingId]/success` - 即座撮影決済完了ページ
- **できること**:
  - 決済完了を確認する

### `/instant/deliver/[bookingId]` - 即座撮影納品ページ
